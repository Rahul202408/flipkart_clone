# Flipkart Clone (MERN Stack)

This project is a Flipkart clone built using MERN Stack.

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js

## Features
- User Login / Signup
- Product Listing
- Add to Cart
- Checkout UI

## Installation

Clone the repository

git clone https://github.com/Rahul202408/flipkart_clone.git

Install dependencies

cd client
npm install

cd ../server
npm install
